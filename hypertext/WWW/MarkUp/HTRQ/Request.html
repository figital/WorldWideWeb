Request elements

DATE

<DATE BEFORE=XXX AFTER=XXX > where the request is a database query, this specifies the before and after dates for the records to be returned.

HEADER

<HEADER> this tag specifies that only the document header (if any exists) is required

LIKE

<LIKE>ZZZ</LIKE> this gives a list of characters used for performing a search (like the '?' criteria in the GET command line)
