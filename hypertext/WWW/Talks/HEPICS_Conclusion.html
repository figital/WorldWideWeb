Availability

Code is available now by anonymous FTP from info.cern.ch [128.141.201.74], directory /pub.

Line Mode Browsers for (almost) any machine. (also HyperText editor for NeXT)

Get a browser, select "Help" then "More" to read all project details from our server. CERN also has a server for all computer centre documents, newsletters, CERN program library documentation, etc. etc.

Anyone can make data available using an FTP server or the HTTP server (available). Several other sites are setting up servers.
