Command Line Syntax

The syntax is www [options] [docaddress [keywords]]
-n
Non interactive mode.
-pn
n is a number which specifies the page length. If there is no number then page size is infinite.
-v
Verbose mode.
-a
Hides positions of anchors, useful when printing out the document.
Any other command line arguments are treated as keywords, but the first argument must correspond to an index. This can be set up by defining an alias for WWW followed by the index name.
