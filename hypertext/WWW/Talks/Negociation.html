Format Negociation

It is negociation between the browsing client and the data supplying server which allows us to cope with the large, growing, number of data formats.

Browser:

Formats acceptable, with penalty points for speed and degradation.

Server:

Data in the best format which is appropriate.

The result is an optimisation of the delay and the degradation of the data, using whatever conversion facilities exist.
