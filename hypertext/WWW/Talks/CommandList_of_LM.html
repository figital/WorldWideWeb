Commands

List
Gives a numbered list of all the links found within the document.
Back
Returns to the previous node.
Home
Returns to the first document read.
Recall
Gives a list of all the documents visited in that session. To select a previous visited document, give the recall command with its number as a parameter.
K
Performs a keyword search, only available in an index.
Help
Gives list of commands.
Quit
Leaves the application.
