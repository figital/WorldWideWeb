Monthly update

On the first of the month, to update the daemon statistics archives, in node INFO,
  • Look at instructions in /usr/adm/Makefile on nxoc01. Modify the makefile as appropriate and then run it. Do this only once (right first time!) or you will overwrite data. Note: the bit for handling the www-log files is not tested.
  • Go to /usr/adm/stats which is a link to a directory on the /W3 disk.
  • Read the instructions in the Makefile there and then run it to generate what you want.
The .dom files are summaries by domain which can be imported into Improv (using row and column names).
