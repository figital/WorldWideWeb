Service Mode

The growing task of providing a real service to users inlcudes the following functions:
Librarian
Provide an overview of the web, and make it as easy as possible to navigate around it and find information. Involves responding to user feedback from searchers and authors, updating keyword indexes and hypertext links in overview documents. A related task is providing help and training to those in physics groups who maintain their own information bases.
Software installation
Providing help to system managers who install WWW software.
Gateway maintenance
Each new field of data which is opened up by a gateway machine requires a small amount fo maintenence. This normally involves ensuring high availability of the gateway, logging the use, and deriving statistics to provide feedback.
