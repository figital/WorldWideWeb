Improvements made to the CERNVM FIND server

In reverse chronological order, these are the improvements which have taken place to the FIND server . Changes are made immediately in machine SERVUS03 (operating occasionally on port 8000), and later transferred to SERVUS02 (operating continually on port 2784).

October 1991

  • Bug fix:Ocasionally the server got into a mode in which it returns all blank lines. This bizarre mode is only reset by killing it.
  • CERN phone book (white pages) on line.

September 91

  • Internet node number of client passed to REXX execs as first parameter for monitoring (26-Sept-91).
  • Log file introduced as in normal daemon.
  • Bug fix: A file containing a line consisting of just slash followed by asterisk put the server into a weird mode in which everything was returned as a sequence of blank lines. (A crazy facet of the IBM C runtime library in which /* was interpreted as eond of file! ) (24 Sep 91)
  • Title added (in C) to seach result virtual documents.

End Mar 91

  • Bug fix: Anchors which do not go anywhere are no longer generated.
  • Bug fix: If nothing is found, a message to that effect is generated, not an acess error message.

11 Feb 91

  • The Yellow pages information and NEWS information has been added.
  • The format has been changed to use heading tags instead of glossary tags.
