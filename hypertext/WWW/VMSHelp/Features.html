VMS Help server Features

This lists the main features of the VMS Help gateway, with improvements in reverse chronological order. Help make it grow fast : send your bug reports and suggestions to Jean-Francois Groff (jfg@cernvax.cern.ch).

Experimental gateway 0.4 -- 2 Oct 91

  • Accepts user queries by number or by name. In the latter case, can go down several levels, for instance, from the main help page : "cc /lib" will go to topic CC, subtopic /LIBRARY.
  • On invocation with only //node:port/HELP, displays the contents of the standard VMS Help library SYS$HELP:HELPLIB.HLB (function lis_to_html).
  • Address format : //node:port/HELP/[@library/][topic[/subtopic]*]
