Installing the Latest Version of LineMode Browser

Update README . See also: LM User guide , administration overview .

Things to do in general:

Test it
Test it on one system!
Compile on all systems
Compile it on each system as below. In WWW/LineMode/xxx, type "make lib" then "make".
Note changes
Note any changes in the change history .
Binaries for FTP
On nxoc01, in WWW/LineMode/xxx, type 'make binary" for EACH machine xxx which has a binary already. (Check with make -n binary first that the binary is up to date)
Sources for FTP
On nxoc01, under /WWW/Library and separately under WWW/LineMode/src, first check the Makefile that any new files have been included (including line mode documentation), then type "make distribute".
ASIS release
On ASIS, do ASIS installation .
Release number
Increment .../Implementation/Version.make by a suitable amount for each project.
Things to do on each system follow. Use the directory hypertext/WWW/PROJECT/MACHINE/ where hypertext is a soft link to the remote mounted cernvax:/userd/tbl/hypertext.
