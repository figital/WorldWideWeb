ON ASIS: updating the repository

ASIS is the software repository. To update things (LMB only at the moment) on ASIS, for each machine architecture, xxxx, first do the make on the appropriate machine , then on ASIS: cd hypertext/WWW/LineMode/Implementation/xxxx nsu c-www make asis ^D This copies the stuff into the right system-specific directory, SO LONG AS the Makefile contains a line defining the ASIS directory. The AIS dirctories have a structure hardware/operating_system, for example /asis/specific/hardware/operating_system/usr.local/bin etc. The bit hardware/operating_system is defined in the Makefile ASIS_MACH = sgi/iris-3.5 which is NOT SET UP YET except for sun4, sgi, rs6000
