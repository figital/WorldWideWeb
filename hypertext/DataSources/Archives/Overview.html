Anonymous FTP
See the (too big) list of sites providing this service, the Archie database of network info, HCI Bibliography , etc etc. Any file available by anonymous FTP can be immediately linked into the web: any WWW browser can use FTP to retrieve it. WWW is ideal for FAQs and lists of FAQ lists. Also available CCITT standards ( Bruno project ). The IETF Anonymous FTP working group (headed by Alan Emtage and Peter Deutsch, famous for Archie ) works on improving access to anonymous FTP areas.

See also: Archie, a program which polls all anonymous FTP sites and makes an index of what they hold.
